# Lowe's (lowes)
Lowe's Companies, Inc. is an American retail company specializing in home improvement. Lowe's operates a developer portal built on Microsoft Azure API Management that provides partners and developers with access to product, inventory, and store APIs for integration with Lowe's retail operations.

## APIs

### Product API
The Lowe's Product API provides programmatic access to Lowe's product catalog, inventory availability, pricing, and store information. Built on Microsoft Azure API Management, the API enables partners and developers to integrate with Lowe's retail operations.
